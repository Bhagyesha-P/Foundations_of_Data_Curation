

# IS547 Foundations of Data Curation 

## Project Title
CURATE(D) Assessment of a Dryad Dataset on Impulsivity and Adverse Childhood Experiences (ACEs)

## Project Description
This repository contains working documentation and progress artifacts for my IS547 data curation project. The project applies the CURATE(D) framework to evaluate a Dryad dataset associated with the study:

**“A study of impulsivity and adverse childhood experiences in a population health setting.”**

The purpose of this project is to assess the dataset’s metadata, documentation, transparency, provenance, and reuse readiness from the perspective of a research data curator.

## Dataset
Dryad Dataset DOI:  
https://doi.org/10.5061/dryad.66t1g1k8x

## Repository Contents
This repository includes:
- Dataset citation
- Metadata assessment
- Repository review notes
- Article review notes
- Article-to-dataset comparison
- Documentation gap notes
- Curation log

## Project Goal
The overall goal of this project is to determine whether the dataset and associated materials are sufficiently curated to support understanding, transparency, and responsible reuse.
